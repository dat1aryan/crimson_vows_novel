## 📖 About the Novel

**Crimson Vows and Obsidian Lies** is a dark romance novel that explores the complex, dangerous relationship between Elara Vance, a talented artist, and Kaelen Thorne, a powerful and enigmatic man with a shadowed past. Inspired by the atmospheric intensity of *God of Malice* and *Haunting Adeline*, this novel delves into themes of obsession, control, trauma, and a love that blurs the lines between desire and danger.

### Story Overview

Elara Vance, a reclusive artist who paints the shadows of the soul, is commissioned by the mysterious Kaelen Thorne to create a piece for his private collection. What begins as a professional arrangement quickly evolves into an intoxicating dance of power, desire, and psychological manipulation. As Elara becomes increasingly entangled in Kaelen's world—a world of luxury, danger, and dark secrets—she discovers that his possessiveness stems from a tragic past and an all-consuming fear of loss.

The novel follows Elara's journey from independence to captivity, from resistance to acceptance, culminating in a climactic confrontation that forces both characters to confront the true nature of their love. With spicy scenes, psychological depth, and an ambiguous, dark resolution, *Crimson Vows and Obsidian Lies* offers a compelling exploration of what happens when two broken souls find each other in the darkness.

### Key Themes

- **Obsession vs. Love:** The blurred lines between possessive control and genuine devotion
- **Trauma and Healing:** How past wounds shape present relationships
- **Darkness Within:** Embracing the shadow side of oneself
- **Power Dynamics:** Control, manipulation, and the surrender of autonomy
- **Redemption and Damnation:** The possibility and impossibility of change

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm (v10 or higher)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/crimson-vows-novel.git
cd crimson-vows-novel
```

2. Install dependencies:
```bash
pnpm install
```

3. Start the development server:
```bash
pnpm dev
```

4. Open your browser and navigate to `http://localhost:3000`

### Building for Production

```bash
pnpm build
```

This creates an optimized production build in the `dist/` directory.

### Preview Production Build

```bash
pnpm preview
```

## 🔧 Customization

### Changing Colors

Edit the color variables in `client/src/index.css`:

```css
:root {
  --primary: #8B1538;           /* Crimson */
  --primary-foreground: #F5F1E8; /* Off-white */
  --background: #0F1419;         /* Deep charcoal */
  /* ... more colors ... */
}
```

### Modifying Typography

Update font imports in `client/index.html` and font assignments in `client/src/index.css`:

```css
body {
  font-family: 'Lora', serif;
}

h1, h2, h3, h4, h5, h6 {
  font-family: 'Playfair Display', serif;
}
```

### Adding New Chapters

Edit `client/src/lib/novelData.ts` and add new chapter objects to the `chapters` array:

```typescript
{
  id: 16,
  title: "Chapter Title",
  content: `Chapter content here...`
}
```
